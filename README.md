## rbl-bi

#### 介绍
这是一个spring boot项目骨架，帮助快速构建应用开发环境和基础结构。

#### 软件架构

使用组件和第三方开源组件：

- SpringBoot
- Druid
- Mybatis Plus
- Swagger-ui
- Knife
- EasyExcel
- Logback
- Lombok

#### 安装教程

1.  git clone ***
2.  使用IDEA打开项目即可
3.  运行BiApiApplication
